# blah-thingy
A C++ 2d game framework built on [Blah](https://github.com/NoelFB/blah) !
This is only a test !

## Features
* Simple C.M.O. implementation
* Sound support
* [Ogmo](https://ogmo-editor-3.github.io/) map support
* Camera
* Graphical debugging tools
